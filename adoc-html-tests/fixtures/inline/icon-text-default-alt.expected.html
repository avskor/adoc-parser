Without :icons:, an inline icon renders as literal bracketed text whose default alt is basename(name).tr('_-', ' ').

A hyphenated icon [fast forward] becomes spaced.

An underscored icon [my cool icon] becomes spaced.

An icon with an extension [foo] drops the extension.

An icon with a path [heart] drops the directory.

A plain icon [heart] stays as-is, and an explicit [Flagged] wins.
